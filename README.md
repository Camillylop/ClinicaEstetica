# Clínica Estética - Clínica Premium

Landing page de alta conversão para clínica de estética avançada, desenvolvida com React, Vite, Tailwind CSS e Shadcn UI.

## Sobre o Projeto

Este projeto foi desenvolvido com uma estética "Clean Girl" e luxuosa, utilizando uma paleta de cores em tons de bege areia, creme e detalhes em dourado fosco.

### Funcionalidades
- **Hero Section**: Apresentação impactante com CTA claro.
- **Tratamentos**: Grid visual de procedimentos.
- **Resultados**: Galeria de antes e depois (simulada).
- **Agendamento Interativo**: Widget passo a passo para marcação de consultas.
- **Experiência**: Vídeo de background e seções informativas.
- **FAQ**: Dúvidas frequentes.

## Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS
- Shadcn UI
- Lucide React (Ícones)
- Framer Motion (Animações)

## Como Executar

1. Clone o repositório
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## Deploy

Para gerar o build de produção:

```bash
npm run build
```
